<AccordionGroup>
  <Accordion title="Can I use my own API key?">
    Yes, you can use your own OpenAI or Gemini API key by adding it in the settings panel.
  </Accordion>
  <Accordion title="What happens when I run out of credits?">
    Each account starts with 50 credits (500,000 tokens). After those are used up, you can either wait for the paid plans to launch or continue using your own API key.
  </Accordion>
  <Accordion title="How do I connect third-party tools?">
    Use webhook URLs in the Functions section to send or receive data through tools like Make.com, Zapier, or any custom backend.
  </Accordion>
  <Accordion title="What models does DacoidChat support?">
    DacoidChat currently supports:

  - DacoidChat native models
  - OpenAI (gpt-4, gpt-3.5)
  - Gemini (via BYOK)

    More models are being added over time.
  </Accordion>
</AccordionGroup>
